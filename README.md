# TylerCaraffi.com

Static personal website for Tyler Caraffi, RCDD.

## Files

- `index.html` — homepage content and SEO metadata
- `styles.css` — responsive visual design
- `script.js` — mobile navigation, year, and subtle reveal effects
- `robots.txt` — search-engine crawl instructions
- `sitemap.xml` — sitemap for Google Search Console

## Publish

Upload these files to the root of the GitHub repository and commit them to the `main` branch.
