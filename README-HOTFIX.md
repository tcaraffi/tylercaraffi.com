# Diagram Hotfix

This package contains only the corrected `index.html`.

## Upload instructions

1. Open the root of the `tylercaraffi.com` GitHub repository.
2. Choose **Add file → Upload files**.
3. Upload `index.html`.
4. Allow it to replace the existing `index.html`.
5. Commit directly to `main` with:
   `Replace infrastructure graphic with responsive SVG`
6. Keep every other file unchanged, especially `CNAME` and the `assets` folder.

The new graphic is an inline SVG, so it does not depend on new CSS classes.
